# Poke Hanoi — Calo Order Web App

Web order poke bowl quét-QR-tại-bàn cho **Poke Hanoi** (11B Hàng Khay). Khách đặt mục tiêu calo, build bowl 6 bước với macro realtime trừ dần, có AI tư vấn dinh dưỡng, đơn chuyển realtime về admin, thanh toán quầy hoặc VietQR.

> HSB F&B Digital Transformation Project — Part 3 (Vibe Coded App). Nhóm 9.

## Tech stack
Next.js 15 (App Router) · TypeScript · Tailwind CSS · Supabase (Postgres + Realtime) · Vercel AI SDK (gpt-4o-mini) · Vercel.

## Bắt đầu
```bash
npm install
cp .env.example .env.local   # rồi điền giá trị thật (xem REQUIREMENTS.md)
npm run dev
```
